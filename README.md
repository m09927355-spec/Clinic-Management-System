# Clinic Management System

## Project Description
This project is a database system designed to manage clinics, doctors, patients, and appointments using MySQL.

## Features
- Department Management
- Clinic Management
- Doctor Management
- Patient Management
- Appointment Scheduling
- Views and Triggers

## Files Included
- create_tables.sql
- load_data.sql
- queries.sql
- views.sql
- triggers.sql

## Tools Used
- MySQL
- GitHub

## How to Run
1. Run create_tables.sql
2. Run load_data.sql
3. Run queries.sql
4. Run views.sql
5. Run triggers.sql
